

# Real Estate 103 — Instructor Course Builder

An all-in-one instructor tool for creating, organizing, and exporting Canvas-ready course materials for RE 103 at TMCC, aligned with Nevada statutes, Pearson VUE exam standards, and your teaching philosophy.

---

## 1. Dashboard & Navigation
- Clean sidebar navigation with access to all major sections
- Overview dashboard showing module completion status and quick actions
- Dark/light mode toggle for comfortable use

## 2. Weekly Module Builder (7-Week Structure)
- Create and organize modules week by week
- Each module follows your required structure:
  - **Key Terms** with definitions
  - **Concept Explanations** in plain English
  - **Nevada Legal References** (NRS/NAC citations)
  - **Real-World Transaction Scenarios**
  - **Common Student Mistakes**
  - **Exam-Focused Key Points** with "Exam Alert" and "High Probability Test Topic" callouts
  - **Knowledge Check Questions** (3–5 per module) with answer explanations
  - **Canvas Discussion Prompt**
  - **Canvas Assignment/Activity suggestion**
  - **Estimated instructional time**
- Drag-and-drop reordering of modules within weeks
- Dual input: manually enter content OR use AI generation (type a topic → get a fully structured module following your course rules and source hierarchy)

## 3. Exam Prep Generator
- Generate Nevada-style multiple choice questions on any topic
- 4 answer options per question with correct answer, explanations, and why wrong answers are wrong
- Tag questions with exam traps, difficulty level, and topic area
- Organize questions into practice exams
- AI generation follows Pearson VUE structural patterns and references Nevada law

## 4. Activity Generator
- Create structured in-class and Canvas activities:
  - Role-plays, case studies, contract drills, closing statement simulations, ethical debates
- Each activity includes instructor notes and debrief prompts
- Tag activities by topic and week for easy placement

## 5. Content Library & Source Tagging
- All content tagged by source authority level (NRS/NAC → Pearson VUE → CE Shop → Lecture Notes → Textbook)
- Visual indicators when content corrects outdated textbook material
- Federal vs. Nevada law distinction clearly marked

## 6. Export System
- **Copy to clipboard**: Formatted content ready to paste into Canvas
- **PDF export**: Download modules, exam prep sets, or activities as formatted PDFs
- **Canvas Quiz Export**: Generate QTI-format files for direct Canvas quiz import

## 7. Design & Experience
- Professional, clean interface with a real estate / academic feel
- Responsive layout optimized for desktop use
- Content preview pane so you can see how materials will look before exporting
- Toast notifications for save/export actions
